# Spacing & Layout System

## Spacing Scale

Our spacing system uses a 4px base unit with a consistent scale:

| Token | Value | Common Usage |
|-------|-------|-------------|
| `space-1` | 4px | Tight inline spacing |
| `space-2` | 8px | Icon-to-text gaps |
| `space-3` | 12px | Input padding |
| `space-4` | 16px | Card padding, list gaps |
| `space-6` | 24px | Section padding |
| `space-8` | 32px | Content area margins |
| `space-12` | 48px | Section separation |
| `space-16` | 64px | Page-level spacing |

## Layout Grid

- **Max width**: 1280px for content areas
- **Columns**: 12-column grid
- **Gutter**: 24px (desktop), 16px (mobile)
- **Margins**: 32px (desktop), 16px (mobile)

## Component Spacing Patterns

### Cards
- Padding: `space-4` (16px)
- Gap between cards: `space-4` (16px)
- Border radius: 8px

### Forms
- Label to input gap: `space-1` (4px)
- Between form fields: `space-4` (16px)
- Form section gap: `space-8` (32px)

### Buttons
- Horizontal padding: `space-4` (16px)
- Vertical padding: `space-2` (8px)
- Gap between buttons: `space-3` (12px)
